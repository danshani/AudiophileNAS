# AudiophileNAS
Linux-based Hi-Res Audio playback system

## Goal
Bit-perfect audio player with automatic file repair and NAS support

## Technologies
- MPD + ALSA
- Python/Flask
- FFmpeg, SoX
- Raspberry Pi

## Structure
- backend/ - Flask API
- frontend/ - Web UI
- features/audio-repair/ - Audio file repair
- nas-integration/ - NAS connectivity
